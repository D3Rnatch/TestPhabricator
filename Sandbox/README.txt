Sandbox
=======

Ce dossier est là pour s'entrainer à git !

Petit exo pour se faire la main :
1) Créer un lien vers ce git sur son pc (ou mac)
2) récupérer la dernière version des fichiers
3) créer un fichier dans le dossier sur son pc
4) le commiter
5) le pusher
6) se reposer (faut pas déconner)

Voir la wiki github pour les commandes, l'utilisation du prog windows, etc.
